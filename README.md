Developed a full-stack movie booking web application (QuickShow) using React, Node.js, and MongoDB with secure authentication via Clerk, enabling users to browse and book movies currently running in their selected location.
◦ Integrated multi-theater functionality, allowing users to choose from multiple theaters for a single movie.
◦ Built an admin dashboard for theater owners to manage theaters, add new shows from a pan-India catalog, and view total bookings and revenue analytics.
◦ Implemented Stripe for secure payment processing, ensuring smooth ticket checkout experience.
